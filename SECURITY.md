# Security Policy

## Supported Versions

This project is currently in early development. Security updates will be provided for the latest version.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability in this project, please report it by:

1. **Do not** open a public issue
2. Email the maintainer directly at the email address in the repository profile
3. Include a detailed description of the vulnerability
4. Provide steps to reproduce if possible

You can expect an initial response within 48 hours. We will work with you to understand and address the vulnerability promptly.
